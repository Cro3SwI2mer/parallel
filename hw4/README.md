# Инструкция по сборке и запуску программы

## Сборка программы

Для сборки программы выполните одну из следующих команд:


```bash
make
```
или

```bash
make all
```

## Запуск программы

### Запуск с параметрами по умолчанию

Запускает программу с 1,000,000 элементов и автоматическим определением числа потоков:


```bash
make run
```
или

```bash
./main
```

### Запуск с пользовательскими параметрами

**Через прямой вызов:**

```bash

# 500,000 элементов
./main 500000


# 2,000,000 элементов
./main 2000000
```

**Через make:**

```bash

# 500,000 элементов
make run-custom ARRAY_SIZE=500000

# 2,000,000 элементов
make run-custom ARRAY_SIZE=2000000
```

### Запуск теста производительности

Запускает серию тестов с разными размерами массивов:

```bash
make benchmark
```

## Очистка собранных файлов

Для удаления скомпилированного исполняемого файла:

```bash
make clean
```

## Второстепенные возможности:

Сборка программы с поддержкой дебаггинг-информации

```bash
make debug
```

Проверка поддержки SIMD-инструкций процессором

```bash
make check-cpu
```

Генерация ассемблерного кода программы

```bash
make assembly
```
