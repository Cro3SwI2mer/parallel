# Инструкция по сборке и запуску программы (задание контрольной работы)

## Сборка программы

Для сборки программы выполните одну из следующих команд:


```bash
make
```
или

```bash
make all
```

## Запуск программы

### Запуск с параметрами по умолчанию

Запускает программу с **n** = 10,000,000:


```bash
make run
```
или

```bash
./main
```

### Запуск с пользовательскими параметрами

**Через прямой вызов:**

```bash

# n = 500,000
./main 500000


# n = 2,000,000
./main 2000000
```

**Через make:**

```bash

# n = 500,000
make run-custom N=500000

# n = 2,000,000
make run-custom N=2000000
```

### Запуск теста производительности

Запускает серию тестов с разными размерами массивов:

```bash
make benchmark
```

## Очистка собранных файлов

Для удаления скомпилированного исполняемого файла:

```bash
make clean
```
