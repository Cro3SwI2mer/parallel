# Инструкция по сборке и запуску программы

## Сборка программы

Для сборки программы выполните одну из следующих команд:


```bash
make
```
или

```bash
make all
```

## Запуск программы

### Запуск с параметрами по умолчанию

Запускает программу с 1,000,000 элементов и автоматическим определением числа потоков:


```bash
make run
```
или

```bash
./main
```

### Запуск с пользовательскими параметрами

**Через прямой вызов:**

```bash

# 500,000 элементов, 2 потока
./main 500000 2


# 2,000,000 элементов, 8 потоков
./main 2000000 8
```

**Через make:**

```bash

# 500,000 элементов, 2 потока
make run-custom ARRAY_SIZE=500000 NUM_THREADS=2

# 2,000,000 элементов, 8 потоков
make run-custom ARRAY_SIZE=2000000 NUM_THREADS=8
```

### Запуск теста производительности

Запускает серию тестов с разными размерами массивов:

```bash
make benchmark
```

## Очистка собранных файлов

Для удаления скомпилированного исполняемого файла:

```bash
make clean
```
