# Инструкция по сборке и запуску программы

## Сборка программы

Для сборки программы выполните одну из следующих команд:

```bash
make
```
или

```bash
make all
```

## Запуск программы

### Запуск с параметрами по умолчанию

Запускает программу с 10,000,000 элементов и автоматическим определением числа потоков:

```bash
make run
```
или

```bash
./main
```

### Запуск с пользовательскими параметрами

Чтобы вызвать краткую документацию/инструкцию к данной программе, выполните следующую команду:

```bash
./main -h
```
или

```bash
./main --help
```

### Запуск теста производительности

Запускает серию тестов:

```bash
make benchmark
```

## Очистка собранных файлов

Для удаления скомпилированного исполняемого файла:

```bash
make clean
```
